## Upgrade Guide

Follow these steps to update SellerClub to the latest version and apply any required changes.

<Steps>
  <Step title="Backup Configuration" icon="database">

    Export your current settings:

````bash
sellerclub config export > backup.json
````

  </Step>
  <Step title="Update Package" icon="download">

    <CodeGroup tabs="npm,yarn">
````bash
npm install sellerclub@latest
````
````bash
yarn add sellerclub@latest
````
    </CodeGroup>

  </Step>
  <Step title="Run Migrations" icon="settings">

    Apply database schema updates:

````bash
sellerclub migrate
````

  </Step>
  <Step title="Restart Service" icon="refresh-cw">

    Restart to activate new features:

````bash
systemctl restart sellerclub
````

  </Step>
</Steps>

<Callout kind="tip">
  After upgrading, verify your integrations in the dashboard at `https://dashboard.example.com/integrations`.
</Callout>
